## 快捷键
* 复制一个副本 `ctrl+D`
* 隐藏网格 `ctrl+``
| ---                                     | ---         |
| --------------------------------------- | ----------- |
| 平移（Pan）                             | 空格 + 拖拽 |
| 缩放至适合屏幕（Zoom to Fit）           | ^ + 1       |
| 缩放至适合选择元素（Zoom to Selection） | ^ + 2       |
| 缩放到 100%（Zoom 100%）                | ^ + 0       |
|                                         |             |
