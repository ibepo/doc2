## 自用常用快捷键

| 快捷键             | 指向                      |
| ------------------ | ------------------------- |
| super+d            | 软件选择                  |
| super+space        | 打开edge                  |
| super+return       | 打开终端                  |
| alt+p              | 截图，flame               |
| spuer+shift+数字键 | 移动当前软件到相应desktop |
| super+数字键       | 切换desktop               |
| super+q            | 关闭当前软件              |
|super+shift+e                    |                          d |
