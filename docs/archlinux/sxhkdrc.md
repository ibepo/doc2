## 自用常用快捷键

### 打开应用
| 快捷键        | 指向            |
| ------------- | --------------- |
| super+d       | 软件选择        |
| super+return  | 打开终端模拟器       |
| super+space   | 打开edge        |
| super+shift+e | 打开资源管理器  |
| super+q       | 关闭当前软件    |
| alt+space     | 打开obsidian    |
| alt+p         | 截图，flameshot |

## 导航

| 快捷键             | 指向                             |
| ------------------ | -------------------------------- |
| spuer+shift+数字键 | 移动当前软件到相应desktop        |
| super+数字键       | 切换desktop                      |
| super+alt+r        | 重启bspwm                        |
| super+escape       | 重启sxhkd                 |
| super+alt+q       | 推出bspwm session                |
| super+a            | 在悬浮窗口和正常窗口之间切换聚焦 |
|                    |                                  |
