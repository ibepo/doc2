# Tmux

## 默认常用操作
- tmux kill-session -a: 删除全部 session(杀会话)
- prefix d : 等价于 tmux detach(暂时逃离)
- prefix & : kill 当前的 window(杀大窗口)
- prefix x : kill 当前的 pane(杀小窗口)
- prefix R : 重新加载配置
