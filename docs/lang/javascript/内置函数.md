## 剩余参数 Rest Parameters
**剩余参数**语法允许我们将一个不定数量的参数表示为一个数组。
语法：
```js
function(a, b, ...theArgs) {
  // ...
}
```
描述：


```js
function sum(...theArgs){
	let total=0;
	for(const arg of theArgs){
		total+=arg;
	}
	return total;
}
console.log(sum(1,2,3))
// Expected output:6

```
