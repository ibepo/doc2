## 04.数据操作 + 数据预处理

### 概念
机器学习和神经网络的主要数据结构是 N维数组

| 表头       | 表头        | 例子                                        |
| -------- | --------- | ----------------------------------------- |
| 0-d （标量） | 一个类别      | 1.0                                       |
| 1-d （向量） | 一个特征向量    | [1.0,2.0,3.0]                             |
| 2-d （矩阵） | 一个样本-特征矩阵 | [[1.0,2.0,3.0][1.0,2.0,3.0][1.0,2.0,3.0]] |
![[Pasted image 20230317100149.png]]
### 创建数组

创建数组需要：
- 形状；例如3x4的矩阵
- 每个元素的数据类型：例如32位浮点数
- 每个元素的值，例如全是0，或者随机数
