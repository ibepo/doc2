## np.vstack

`np.vstack` 是NumPy中的一个函数，它会将一组数组在垂直方向（按行）合并。它的输入参数是一个数组组成的元组或列表，输出是一个包含所有输入数组的垂直合并的新数组。

``` python
import numpy as np
x = np.array([1, 2, 3])
y = np.array([4, 5, 6])
# 将 x 和 y 垂直堆叠 
z = np.vstack((x, y))
print(z)
-----------------------------
[[1 2 3]
 [4 5 6]]
```

在这个例子中，`np.vstack((x, y))` 就是将 `x` 和 `y` 按垂直方向堆叠，生成一个新的数组 `z`，其中每行分别包含了 `x` 和 `y` 的元素值。
